# 資料型態
## 資料型態的用途
宣告一變數或函式時 需要在程式中註明資料型態

1. 程式可方便分配物件其記憶體 及約束其物件使之在可控的情況下可執行出預期的結果

2. 程式開發者可依物件型態判斷出物件的用途等

( 在程式語言中會分強型別與弱型別 C++為強型別語言之一 )

## 常見的資料型態
| 型態名稱 | 型態說明  | 資料大小 |
| -------- | -------- | --------|
| int      | 整數     | 8 bytes |
| float    | 浮點數   | 1 bytes |
| char     | 字元     | 1 bytes |
| boolean  | 布林值   | 1 bytes |
`註: 1 byte 為 8 bits`
